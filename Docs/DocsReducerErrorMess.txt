Обычно эта ошибка возникает в том случае, если для открываемого файла установлена блокировка чтения. Это может быть вызвано следующими причинами:
Файл открыт другим пользователем на этом или другом компьютере.
При возникшем ранее сбое приложения Word была установлена блокировка чтения файла.
Файл заблокирован другим приложением, это не позволяет открыть его в редакторе Word.
Запущено пользовательское приложение, которое открыло этот файл (может быть, на другом компьютере). Возможно, файл был открыт с помощью неправильного метода.
Для устранения неполадки закройте все приложения, повторно запустите редактор Word и попробуйте снова открыть файл. Если это не помогает, проверьте свойства файла - может быть, он уже открыт другим пользователем. Изредка случается, что файл заблокирован мнимым процессом (неработающей или отсоединенной программой). В этом случае единственный способ устранения неполадки - перезагрузка компьютера.
